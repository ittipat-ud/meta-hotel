# 🏨 META HOTEL — Booking System Website

เว็บไซต์จองห้องพักระดับพรีเมียมของโรงแรม **META** หน้าตาสวยหรูหรา ใช้งานง่าย รองรับสองภาษา (ไทย/อังกฤษ) พร้อมระบบหลังบ้าน Admin Dashboard สำหรับติดตามสถิติสถานะห้องพัก ค้นหาประวัติผู้จอง และส่งออกข้อมูลเป็น CSV โดยเชื่อมต่อฐานข้อมูลไร้เซิร์ฟเวอร์ด้วย Google Sheets + Apps Script

---

## ✨ คุณสมบัติหลัก (Key Features)

1. **การออกแบบพรีเมียมระดับห้าดาว (Premium Visual Experience):**
   - โทนสี **น้ำเงินเข้ม (Royal Blue), ดำเข้ม (Charcoal Gray) และ ขาวสะอาด** เพื่อความเรียบหรูและอ่านง่าย
   - พื้นหลังแบบกระจกฝ้า (Glassmorphism) และการจัดหน้าแบบ Responsive รองรับหน้าจอคอมพิวเตอร์ แท็บเล็ต และมือถือ
   - ไมโครแอนิเมชันและการแสดงผลแบบค่อยๆ เลื่อนปรากฏ (Scroll Reveal) เมื่อผู้ใช้อ่านเนื้อหา

2. **ฟังก์ชันการจองที่แม่นยำ (Accurate Room Booking):**
   - คำนวณราคาแบบเรียลไทม์ (Real-time Dynamic Price Summary) คูณตามจำนวนคืนและตัวเลือกเสริม
   - ประเภทห้องพัก 2 แบบหลัก:
     - **Standard Room:** พักได้ 2 คน (ราคา 1,000 บาท/คืน)
     - **Family Room:** พักได้ 4 คน (ราคา 2,000 บาท/คืน)
   - เลือกประเภทเตียงเดี่ยว หรือ เตียงคู่ และเลือกบริการเสริมเตียงเสริม (+500 บาท/คืน)
   - ป้อนชั้นพักและห้องพักที่ชัดเจน (โรงแรม 10 ชั้น ชั้นละ 10 ห้อง รวม 100 ห้อง)
   - ตรวจสอบความถูกต้องของข้อมูล (Form Validation) เช่น ตรวจความยาวชื่อเล่น รูปแบบเบอร์โทรศัพท์ และป้องกันการเลือกวัน Check-out ย้อนหลัง Check-in

3. **ระบบสลับภาษา (i18n Localization):**
   - เปลี่ยนภาษาของหน้าเว็บเป็น ภาษาไทย (TH) และ ภาษาอังกฤษ (EN) ได้อย่างรวดเร็วที่แถบนำทางด้านขวาบน
   - แปลข้อความรวมถึงข้อความแจ้งเตือนความผิดพลาด (Validation Alert) และตัวเลือกฟิลด์วัน-เดือน-ปี โดยใช้ Attribute `data-i18n`

4. **แผงจัดการผู้ดูแลระบบ (Admin Dashboard):**
   - เข้าสู่ระบบด้วยรหัสผ่านความปลอดภัย (รหัสเริ่มต้น: `meta2026`)
   - **หน้าสถิติหลัก (Stats Indicators):** แสดงยอดจองทั้งหมด, ห้องว่าง, รายได้รวมที่คาดการณ์ และ ยอดจองใหม่ในวันนี้
   - **แผนผังผังอาคาร (Occupancy Floor Map Grid):** แสดงภาพสีห้องที่จองแล้ว (สีแดง) และห้องว่าง (สีเขียว) แยกตามชั้น 1-10 ของโรงแรม
   - **ตารางสรุปข้อมูล (Interactive Data Table):** ค้นหารายการผ่าน Search Box (ค้นจาก ชื่อ, เบอร์โทร, ห้อง) เรียงตามวันจองล่าสุด และสนับสนุนการ Export ข้อมูลทั้งหมดเป็นไฟล์ CSV (ดาวน์โหลดรองรับตัวอักษรภาษาไทย ไม่พบปัญหาฟอนต์เพี้ยนด้วย UTF-8 BOM)

5. **สถาปัตยกรรมแบบขยายได้ (Modular Plugin Architecture):**
   - แยกโครงสร้างโค้ดและดีไซน์อย่างชัดเจนเพื่อให้นักพัฒนาคนอื่นนำไปต่อยอดได้ง่าย (เช่น การเพิ่มฟีเจอร์ Chat Box ผ่านโฟลเดอร์ `js/plugins/`)

## 🚀 วิธีการติดตั้งและรันโปรเจกต์

1. **เตรียมไฟล์รูปภาพ:**
   - โฟลเดอร์ `assets/images/` มีไฟล์ภาพที่ถูกสร้างไว้เรียบร้อยแล้ว ได้แก่ `hero-bg.png`, `standard-room.png`, และ `family-room.png`

2. **เปิดระบบทดลองใช้งานบนเครื่องคอมพิวเตอร์:**
   - โปรเจกต์นี้เขียนด้วย HTML/CSS/Vanilla JavaScript โดยไม่มีการใช้เฟรมเวิร์กที่ยุ่งยาก คุณสามารถรันหน้าเว็บได้ทันทีด้วย:
     - การติดตั้งปลั๊กอิน **Live Server** ในโปรแกรม VS Code
     - หรือเปิดไฟล์ `index.html` บนเบราว์เซอร์ตรงๆ
   - *หมายเหตุ:* ในโหมดเริ่มต้น หากยังไม่ได้ระบุ `API_URL` โครงสร้างจะทำงานผ่าน **Mock Mode** (บันทึกข้อมูลจองจำลองลงใน LocalStorage ของบราวเซอร์ผู้ใช้โดยอัตโนมัติ ทำให้ผู้ใช้เปิดหน้า `admin.html` เพื่อดูสถิติและทดสอบระบบได้โดยไม่ต้องตั้งค่าฐานข้อมูลทันที)

3. **ตั้งค่าฐานข้อมูล Google Sheets:**
   - หากพร้อมที่จะใช้งานจริง ให้ตั้งค่าตามขั้นตอนในคู่มืออย่างละเอียดที่ไฟล์ [SETUP.md](SETUP.md)

## 🔌 คู่มือนักพัฒนา (Developer Tweak Guide)

### 1. วิธีเพิ่มภาษาใหม่
1. เปิดไฟล์ `js/i18n.js`
2. ค้นหา `this.translations = { ... }`
3. เพิ่มอ็อบเจกต์ภาษาใหม่ เช่น ภาษาจีน `zh`:
   ```javascript
   zh: {
     'nav.home': '首页',
     'nav.about': '关于เรา',
     // ...
   }
   ```
4. เพิ่มปุ่มสลับภาษาในไฟล์ `index.html` ภายในบล็อก `<div class="lang-switcher">` :
   ```html
   <button class="lang-btn" data-lang="zh">ZH</button>
   ```

### 2. วิธีการแก้ไของค์ประกอบราคาหรือข้อมูลโรงแรม
คุณสามารถเปลี่ยนราคา รายละเอียดห้องพัก หรือรหัสผ่านผู้ดูแลระบบได้ง่ายๆ ที่ไฟล์ `js/config.js`:
- หากต้องการเปลี่ยนค่าบริการเสริมเตียงเสริม ให้ปรับปรุงตัวเลขที่คีย์ `PRICING.EXTRA_BED`
- หากต้องการเปลี่ยนรหัสผ่านแผงแอดมิน ให้ปรับปรุงที่คีย์ `ADMIN_PASSWORD`
