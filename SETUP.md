# 📊 SETUP.md — คู่มือเชื่อมต่อ Google Sheets + Apps Script

ไฟล์นี้อธิบายถึงขั้นตอนการสร้างแอปหลังบ้านเพื่อบันทึกข้อมูลการจองห้องพักลงใน Google Sheets โดยไม่ต้องใช้เซิร์ฟเวอร์ส่วนตัว (Serverless Database)

---

## 🛠️ ขั้นตอนการตั้งค่าระบบ

### 1. สร้าง Google Sheet และคัดลอกรหัสชีท
1. เปิด [Google Sheets](https://sheets.google.com) และกดสร้าง **สเปรดชีตเปล่า (Blank Spreadsheet)**
2. ตั้งชื่อไฟล์ด้านซ้ายบนตามใจชอบ เช่น `META Hotel Bookings`
3. สังเกตแถบ **URL ด้านบน** ของบราวเซอร์เพื่อความเข้าใจโครงสร้าง (ไม่ต้องใช้รหัส ID ของชีทโดยตรง เนื่องจาก Apps Script จะผูกกับชีทแบบแอ็กทิฟให้อัตโนมัติ)

### 2. นำโค้ด Apps Script ไปติดตั้ง
1. บนแถบเมนูด้านบนของ Google Sheet ให้คลิกเมนู **ส่วนขยาย (Extensions) > Apps Script**
2. หน้าต่างการเขียนโค้ดจะแสดงขึ้นมา มีไฟล์เริ่มต้นชื่อ `รหัส.gs` (หรือ `Code.gs`)
3. ลบโค้ดเริ่มต้นออกให้หมด
4. ไปที่ไฟล์ [google-apps-script/Code.gs](google-apps-script/Code.gs) คัดลอกโค้ดทั้งหมดแล้วนำมาวางในหน้าเขียนโค้ด
5. กดไอคอน 💾 (บันทึกโครงการ) ด้านบน หรือกด `Ctrl + S`

### 3. เริ่มต้นการ Deploy เป็น Web App
1. กดปุ่มสีน้ำเงิน **การใช้งานได้จริง (Deploy) > การจัดการการปรับใช้งานใหม่ (New deployment)**
2. คลิกรูปเฟือง ⚙️ (เลือกประเภทการปรับใช้งาน) ข้างคำว่า "เลือกประเภท" แล้วเลือก **เว็บแอป (Web app)**
3. ตั้งค่ารายละเอียดดังนี้:
   - **คำอธิบาย (Description):** เช่น `META Hotel Booking API v1`
   - **เรียกใช้งานเป็น (Execute as):** เลือก **ฉัน (อีเมลของคุณ)**
   - **ผู้มีสิทธิ์เข้าถึง (Who has access):** เลือก **ทุกคน (Anyone)** *(สำคัญมาก! หากไม่เลือกหัวข้อนี้ หน้าเว็บบนเบราว์เซอร์ของลูกค้าจะไม่สามารถกดบันทึกข้อมูลจองลงชีทได้)*
4. กดปุ่ม **ปรับใช้งาน (Deploy)**

### 4. อนุญาตสิทธิ์การเข้าถึงข้อมูล (Authorization)
1. ในขั้นตอนการ Deploy ครั้งแรก จะมีป๊อปอัปให้ยืนยันตัวตน ให้คลิก **อนุญาตการเข้าถึง (Authorize access)**
2. เลือกบัญชี Google ของคุณ
3. หากแสดงหน้าแจ้งเตือนว่า "Google ไม่ได้ตรวจสอบแอปนี้" ให้กดลิงก์ **ขั้นสูง (Advanced)** ที่อยู่ด้านล่างซ้าย
4. คลิกลิงก์ **ไปที่ META Hotel Booking... (ไม่ปลอดภัย) / Go to META Hotel Booking... (unsafe)**
5. กดปุ่ม **อนุญาต (Allow)**

### 5. คัดลอกลิงก์ Web App URL ไปใส่ใน config
1. เมื่อกระบวนการปรับใช้งานสำเร็จ ระบบจะแสดง **URL ของเว็บแอป (Web app URL)**
   - URL จะมีลักษณะคล้าย: `https://script.google.com/macros/s/AKfycb.../exec`
2. คัดลอก URL นั้นไว้
3. เปิดไฟล์โปรเจกต์ `js/config.js` บนเครื่องคอมพิวเตอร์ของคุณ
4. หาบรรทัด `API_URL: 'YOUR_GOOGLE_APPS_SCRIPT_URL_HERE',` แล้วนำลิงก์ที่คัดลอกมาวางแทนที่
   ```javascript
   // ตัวอย่างการตั้งค่าที่ถูกต้อง
   API_URL: 'https://script.google.com/macros/s/AKfycbxXXXXXXXXXXXX-XXXXXXXXXXXXX/exec',
   ```
5. บันทึกไฟล์ `js/config.js` เป็นอันเสร็จสิ้นขั้นตอน!

---

## 🔍 วิธีการทดสอบการเชื่อมต่อ
1. เปิดหน้าเว็บ `index.html` บนบราวเซอร์ของคุณ
2. กรอกฟอร์มจองห้องพัก เช่น พิมพ์ชื่อเล่น `น้องเป็ด` เลือกเตียงเสริม กรอกวันที่ แล้วกด **ยืนยันการจอง**
3. หากสำเร็จ จะแสดงป๊อปอัปแจ้งเตือนว่า **จองสำเร็จ! 🎉**
4. ย้อนกลับไปเปิดหน้า Google Sheets แผ่นงานชื่อ **Bookings** จะถูกสร้างขึ้นมาพร้อมข้อมูลแถวใหม่ที่กรอกไปทันที!
5. ทดสอบเปิด `admin.html` เข้าสู่ระบบด้วยรหัสผ่าน `meta2026` หน้าประวัติการจองและแผนผังห้องพักจะแสดงข้อมูลที่เชื่อมมาจาก Google Sheet โดยอัตโนมัติ!
